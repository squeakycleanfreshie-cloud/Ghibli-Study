# Ghibli Study Room

A simple study web app with a task list, progress tracking, and a timer. Designed with a calm, Studio Ghibli-inspired aesthetic.

# Features
Add and complete tasks
Progress bar updates based on completed tasks
Study timer with start and reset controls
Simple responsive layout with background image support

# How to use
Open index.html in a browser and start adding tasks. Use the timer to track study sessions.
